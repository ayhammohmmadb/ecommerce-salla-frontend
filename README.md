# Ecommerce Salla Frontend

A modern eCommerce frontend built using React and Vite.  
This project provides a fast and responsive shopping experience with a clean UI and dynamic product pages.

## 🚀 Features

- Responsive modern design
- Product listing and categories
- Product details page
- Shopping cart functionality
- React Router navigation
- Loading animations
- Swiper sliders
- Dynamic API fetching

## 🛠️ Technologies Used

- React
- Vite
- React Router DOM
- CSS3
- Swiper.js
- React Icons
